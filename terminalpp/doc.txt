ARGS:
"--version - mostra a versão do Terminal++"
"--version -s - verifica a versão pelo servidor e mostra se está atualizado"
"--version -k - mostra a key do Terminal++"
"--about - mostra a descrição do Terminal++"
"--enabled - mostra se o Terminal++ está funcionando corretamente"
"--check - mostra o caminho do Terminal++"
"--size - mostra o tamanho do executável do Terminal++ em bytes"
"--size -kb - mostra o tamanho do executável em KB"
"--size -mb - mostra o tamanho do executável em MB"
"--size -gb - mostra o tamanho do executável em GB"
"--size -by - mostra o tamanho do executável em bytes (padrão)"
"--docs - mostra a documentação do Terminal++"
"--cie <NOME> - busca informações externas pelo nome (ex: raws do GitHub)"
"--ut <COMANDO> - executa um comando do Terminal++ direto pelos argumentos"

COMANDOS TERMINAL:
"exit - fecha o Terminal++"
"killtpp - força o fechamento do Terminal++"
"check_infos / infos - mostra informações do usuário, sistema e CPU"
"cd <CAMINHO> - muda o diretório atual"
"all_errors / check_errors / errors - mostra todos os erros do sistema"
"credits - mostra os créditos do Terminal++"
"^><COMANDO> - executa um comando diretamente no sistema operacional"
"open <APP> - abre um aplicativo no Windows usando o comando start"
"say <MENSAGEM> - mostra uma mensagem no terminal"
"check_updates / updates - verifica se o Terminal++ está atualizado"
"check_key / key - mostra a key do Terminal++"
"check_docs / docs - mostra a documentação"
"check_admin / check_adm / adm / admin - verifica se o Terminal++ está rodando como administrador"
"check_info_ext <NOME> - busca informações externas pelo nome"
"ext <NOME> - atalho para check_info_ext"
"check_tpp / check_t++ / check_terminal / check_terminal++ - mostra o caminho do Terminal++"
"otc <true|false> - ativa ou desativa comandos do sistema operacional"
"dir - lista arquivos do diretório atual"
"&path - adiciona o Terminal++ ao PATH do sistema"
"&rpath - remove o Terminal++ do PATH do sistema"
"del_path <CAMINHO> - remove um caminho do PATH"
"add_path <CAMINHO> - adiciona um caminho no PATH"
"verify_tpp / verify_t++ / verify_terminal / verify_terminalpp / verificy_terminal++ - verifica se o Terminal++ está corretamente configurado"
"check_size <CAMINHO> - mostra o tamanho de um arquivo"
